Создать игровой аппарат.
Создать проект на spring data jdbc + spring rest.
Создать в базе данных таблицу finance_result (income, outcome).
В этой таблице будут накапливаться все доходы и расходы игрока
в одной строчке. income - доходы игрока, outcome - расходы игрока
Создать в базе данных таблицу game (id, 1_sym, 2_sym, 3_sym).
В этой таблице будут результаты по каждой игре.
Создать endpoint /play.
Данный метод будет:


 1 открывать транзакцию.
 2 В finance_result в outcome добавляется 15.
 3 Генерится 3 символа.
	Есть 3 символа (A, F, D), при выпадении дают след доход:
	А А А - 10
	F F F - 20
	D D D - 50
	Если выпадает A F D - ход бесплатный.
 4 Если выпадает A F D - транзакция откатывается. Возвращается
 http ответ: Бесплатный ход.
 5 Есил не A F D и выпадает комбинация, то в income заносится
 сумма выигрыша. Возвращается http ответ: Вы выиграли - {сумма выигрыша}.
 6. Если комбинация не выпадает. Возвращается http ответ: Вы ничего не выиграли.

Создать endpoint /history. Данный метод возвращает ответ в следующем формате:
{
 "playerIncome": значение income из finance_result,
 "playerOutcome": значение outcome из finance_result,
 "game_history": [результаты последних 5 игр]
}
